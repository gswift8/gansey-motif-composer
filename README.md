# Gansey Studio v0.9.3 — Drag-and-Drop Reordering Repair

This release restores drag-and-drop reordering for motifs and spacers.

## Improvements

- Every motif and spacer card now has a visible **⠿ Drag** handle.
- Drag left or right within the same band.
- Drag a block to another vertical or horizontal band.
- Drop on the left or right half of a card to control placement.
- Drop into empty space at the end of a band to append it.
- Clear before/after insertion indicators.
- Move buttons remain available as an accessible fallback.
